Parágrafos e quebras de linhas

A ideia é mostrar as quebras de linhas usando elementos do html como as tags

Ainda podemos usar os sinais de > e < sem acionar uma tag mesmo usando as siglas dessas mesmas tags, Por exemplo usando br você tem a quebra de linha, mas se usar os sinais de maior e menor ela quebra a linha nesse ponto se a ideia é mostrar em formato de texto podemos usar < e > colocando no meio qualquer sigla usanda nas tags, por exemplo <br> usado para quebra de linha como dito anteriormente
